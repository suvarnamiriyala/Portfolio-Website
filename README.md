## How to Customize

### Update Personal Information
1. Open `index.html`
2. Replace the example content with your actual information:
   - Change "Suvarna Miriyala" to your name
   - Update the About section with your bio
   - Add your actual projects
   - Update your skills
   - Add your real contact information

### Customize Colors
1. Open `styles.css`
2. Locate the `:root` section at the top
3. Update the CSS variables:
   ```css
   :root {
       --primary-color: #2c3e50;      /* Main color */
       --secondary-color: #3498db;    /* Accent color */
       --accent-color: #e74c3c;       /* Highlight color */
       /* ... etc */
   }
   ```

### Add Project Links
Update the project card links in the HTML:
```html
<a href="your-project-link" class="btn btn-secondary">View Details</a>
```

### Update Contact Links
Replace the example contact information:
```html
<a href="mailto:your-email@example.com" class="contact-link">Email</a>
<a href="https://linkedin.com/in/your-profile" class="contact-link">LinkedIn</a>
<a href="https://github.com/your-username" class="contact-link">GitHub</a>
```
